# TweetX (Twitter Like Web App made using PERN Stack)

## Technology Used For Frontend.
- [TypeScript](https://www.typescriptlang.org/)
- [React](https://reactjs.org/)
- [Zustand for react state management](https://zustand-demo.pmnd.rs/)
- [React Router Dom@6 for browser routing](https://reactrouterdotcom.fly.dev/)
- [TailwindCSS for styling](https://tailwindcss.com/)
- [Vite for frontend tooling](https://vitejs.dev/)
